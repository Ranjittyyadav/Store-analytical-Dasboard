# 📊 Store Sales Analytics Dashboard

Interactive Excel dashboard analyzing **₹21M+** in 2022 Indian ethnic wear sales across **31K+ orders** from Amazon, Myntra, Flipkart, Ajio & more.

![Dashboard Preview](screenshots/dashboard.png)

## 🔗 Dataset Source
Based on anonymized 2022 Indian e-commerce sales data similar to:  
[![Kaggle](https://img.shields.io/badge/Kaggle-E--Commerce%20Sales-blue)](https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data)

## 🚀 Quick Start

1. **Download** `Store-Data.xlsx` (~5.6MB)
2. **Open in Excel** (2016+ recommended) 
3. **Enable content** when prompted
4. **Navigate tabs** - slicers work instantly!

## 📋 Features

- ✅ **8 Interactive pivot dashboards**
- ✅ **Live slicers** - Month/State/Gender/Channel filters
- ✅ **No VBA required** - Pure Excel Power
- ✅ **Auto-refresh** pivots on data changes
